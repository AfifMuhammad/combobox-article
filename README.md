# Custom Combobox with Remote Data

This project is the companion code for the Medium article: [Building a Custom Combobox with Remote Data in React using shadcn/ui and TypeScript](https://afifm.medium.com/building-a-custom-combobox-with-remote-data-in-react-using-shadcn-ui-and-typescript-0f848bc71022).

It demonstrates how to build a reusable combobox component that fetches remote data with pagination, using [Next.js](https://nextjs.org), [shadcn/ui](https://ui.shadcn.com), and TypeScript.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
